GNATS v3.113 patches

This page collects patches for GNATS v3.113.

  • Buffer overrun when number of classes in the classes file becomes too large
  • Missing space in Date: header
  • Fixes for documentation typos

Buffer overrun when number of classes in the classes file becomes too large

This patch is to fix a buffer overrun problem in files.c in the function get_class_enum_field() which can cause a crash when the number of classes in the classes file becomes too large. Submitted by Marcelle Gannon. Download patch.

Missing space in Date: header

File-pr sends out copies of PRs with a missing space after Date: in the mail headers, like "Date:Mon, 15 Jan 2001 12:53:47 +0100". This causes problems for many mail clients. Submitted by Yngve Svendsen. Download patch.

Fixes for documentation typos

The following patches fix typos in the GNATS 3.113 documentation. Submitted by Yngve Svendsen.

Download patch for: gnats.texi, fields.texi, p-admin.texi, p-inst.texi, p-usage.texi.
