> Should I run with that, or is there a file or set of instructions on
> setting up GNATS 4.0 ? I need the flexibility of 4.0 (GNATS 3.* is too

Have a look in the man pages, particularly dbconfig, databases and
gnatsd. Anything that you can't find in there you'll have to go to the
source for. Setting up gnatsd is the same as version 3, as are the
format and contents of most of the files in the gnats-adm directory so
the old doco may help there.

If you have any further questions, post them here and we'll see what
we can do. Be aware though that 4.0 is (barely) beta software.
