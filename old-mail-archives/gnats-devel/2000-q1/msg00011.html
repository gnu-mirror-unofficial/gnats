gnats2w 0.14 released

  • To: GNATS Development <gnats-devel at sourceware dot cygnus dot com>
  • Subject: gnats2w 0.14 released
  • From: Milan Zamazal <pdm at freesoft dot cz>
  • Date: 24 Jan 2000 22:03:16 +0100
gnats2w 0.14 is available at
ftp://ftp.debian.org/debian/dists/woody/main/source/web/gnats2w_0.14.2.tar.gz

It has the following new features:

- Work with more databases fixed.
- States and classes are not static anymore, but gotten from query-pr.
- Few new pieces of documentation written.
- Other bug fixes.

New features from TODO might be added in future (especially gnatsd
support, but not before Gnats 4 is released) if I get enough interest
indication for them (I don't need them myself now).

(gnats2w is yet another Web interface to GNATS, written in Python.)
