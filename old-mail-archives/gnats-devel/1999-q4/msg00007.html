I've moved the gnats/contrib directory up to the same level as the gnats/
directory in the CVS repository.

To update your sources, you should do the following at the top-level of
your GNATS CVS checked out tree:


 cvs update -P -d


This command will check out the new contrib/ directory and remove the
old contrib/ directory.

There aren't any changes to the files in the contrib/ directory at this
time -- I've just moved them.

If you're not currently using CVS to get at the current GNATS sources,
you can ignore this.

Contact me if you see any problems.

Jason
